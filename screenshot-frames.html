📱 How to Export at App Store Quality (1284 × 2778 px)

  1. Open Chrome DevTools (Cmd+Option+I)
  2. Click the device toolbar icon (top left of DevTools)
  3. Set device to "iPhone 14 Plus" — or manually set 390×844 at 3× DPR
  4. Close DevTools, reload the page
  5. Right-click each frame → Inspect → Screenshot node or use the Capture node screenshot command from DevTools
  6. Replace placeholder screens: each <img> has a data-screen comment — swap the src with your real simulator screenshot
